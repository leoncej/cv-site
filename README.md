# cv-site
A fully client-side, single-page CV website built with HTML, CSS, and JavaScript, designed to present professional experience in a code-styled interface inspired by modern IDEs. The site is responsive, lightweight, and requires no backend or build tooling.

Technology Stack: HTML5: Semantic markup and structural layout, CSS3: Custom styling, responsive grid layout, and IDE-inspired theme, Vanilla JavaScript (ES6): DOM manipulation, scroll tracking, and minimap rendering, Architecture: Static site (no backend), Single HTML entry point, Inline styles and scripts for portability and ease of deployment
